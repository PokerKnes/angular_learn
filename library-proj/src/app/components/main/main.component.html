Главная страница

Мое первое, кривое, косое приложение на ангуляре. Что-то типо библиотеки. Поиск книг осуществляется по открытому api gutendex. А на странице с добавленными книгами есть немного логики с фильтрацией, удалением, комментариями. Данные нигде не хранятся, так что после обновления страницы все исчезнет. Может быть потом доделаю бэк или хотя бы запихну книги в local storage.
